Caso de Uso – Busca serviço

Identificador

UC001

Nome

Busca serviço

Atores

Primário: Cliente
Secundário: Funcionario

Sumário

O cliente procura um serviço para o seu pet e agenda um horário para realizar no pet shop

Complexidade

N/A

Regras de Negócio

RN1, RN2, RN3

Pré-condições

O cliente tem conta registrada no sistema

Pós-condição

O serviço agendado foi solicitado para o funcionário

Pontos de Inclusão

N/A

Pontos de Extensão

N/A

Fluxo Principal:

Fluxo Principal

Ações do Ator

Ações do Sistema

  1. O cliente solicita ver os serviços
  1. O sistema lista os serviços
  1. O cliente seleciona um serviço
  1. O sistema solicita a ficha do pet registrado no sistema
  1. O cliente seleciona a ficha do pet
  1. O sistema lista os locais disponíveis para realizar o serviço
  1. O cliente seleciona o local
  1. O sistema apresenta o período de funcionamento do serviço
  1. O cliente seleciona a data e o horário dentro do período disponível
  1. O sistema lista as informações passadas para o cliente revisar
  1. O cliente confirma o pedido
  1. O sistema envia as informações para o funcionário

Fluxos alternativos:

Fluxo Alternativo – Passo 4 - condição

Ações do Ator

Ações do Sistema

  1. No Passo 5, o pet do cliente não está registrado no sistema
  1. O sistema solicita informações para registrar o pet
  1. No passo 11, o cliente solicita mudar o agendamento
  1. O sistema volta para o passo 8
  1. No passo 11, o cliente solicita mudar o local
  1. O sistema volta para o passo 6
  1. No passo 11, o cliente solicita mudar a ficha do pet
  1. O sistema volta para o passo 4
  1. No passo 11, o cliente solicita mudar o serviço
  1. O sistema volta para o passo 2

Fluxos de exceção

Fluxo de Exceção: Passo 2 (exemplo)

Ações do Ator

Ações do Sistema

  1. No passo 5, o cliente seleciona mais de duas fichas
  1. O sistema permanece no passo 4
  1. No passo 9, o cliente seleciona um período indisponível
  1. O sistema permanece no passo 8
  1. No passo 11, o cliente recusa o pedido
  1. O sistema volta para o passo 8

Regras de Negócio Casos de Uso

Número

Regra de Negócio

RN1

O cliente precisa selecionar apenas um pet por pedido

RN2

O cliente precisa selecionar uma data e horário dentro do período de funcionamento do serviço

RN3

O cliente precisa confirmar o pedido
